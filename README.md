 # My-Tibabu Health Connect

A comprehensive healthcare platform connecting patients with doctors.

## Project Structure

```
tibabu-health-connect/
├── backend/          # Node.js/Express backend
├── client/           # React/Vite frontend
└── README.md
```

## Getting Started

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. Create a `.env` file with your configuration:
   ```env
   PORT=3002
   JWT_SECRET=your_jwt_secret_here
   MONGODB_URI=your_mongodb_connection_string
   ```

4. Start the backend server:
   ```bash
   npm start
   # or
   pnpm start
   ```

### Frontend Setup

1. Navigate to the client directory:
   ```bash
   cd client
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

The frontend will be available at `http://localhost:8080`

## Environment Variables

### Backend (.env)
- `PORT`: Server port (default: 3002)
- `JWT_SECRET`: Secret key for JWT tokens
- `MONGODB_URI`: MongoDB connection string

### Frontend (.env)
- `VITE_API_URL`: Backend API URL (default: http://localhost:3002/api)
- `VITE_SOCKET_URL`: WebSocket URL (default: http://localhost:3002)
